some tools I've built

A couple years ago, I set out to learn some coding & programming to make my life easier by automating boring/repetitive tasks & building mini-scripts that would eliminate the need for downloading bloated software or programs that compromise privacy.
  • screenshot app : The ones I found online force you to first save the screenshot to a directory before being allowed to take another one. And I wanted something like we have in mobile : images get saved automatically and you can take a screenshot every second. So i've built this one using Python and Tkinter.
  • Voice activated timer : A simple, voice-activated timer that you can use directly in your browser. It is designed to help you easily set and manage rest periods while working out (or for any other activity where a countdown is needed and you can't use your hands). An alert sound plays during the last 4 seconds of the countdown to ensure you are notified before time expires.
